## Project Descriptions

### 01 · Customer Segmentation \& Cohort Analysis

**Role:** Data Analyst  ·  **Dataset:** UCI Online Retail II  ·  📊 [Tableau Dashboard](https://public.tableau.com/app/profile/dilsha.alex/viz/CustomerSegmentationCohortAnalysisDashbaord/Dashboard)

Retail businesses lose a significant portion of their customer base annually, yet most marketing budgets treat all customers identically. This project aims to segment customers by purchasing behaviour using RFM (Recency, Frequency, Monetary) analysis and cohort retention tracking, enabling a retail business to identify its highest-value customers, those at risk of churning, and those already lapsed, so that marketing spend and retention strategies can be allocated with precision rather than uniformly.

## How to Use This Repository

Each project folder is self-contained. To run any project:

```bash
# 1. Navigate to the project folder
cd 01\_customer\_segmentation

# 2. Install dependencies
pip install -r requirements.txt

# 3. Follow the running instructions in the project's README.md
```

All running instructions, dataset download links, and setup steps are documented inside each project's own `README.md`.
